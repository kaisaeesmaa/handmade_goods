# handmade_goods

This is a simple notes app built with Vue and Express. It uses plain json array to store notes.

# Project management

https://www.pivotaltracker.com/n/projects/2634045

## Project setup

``` 
cp .env.example .env
``` 

``` 
npm install
```

### Running the app
```
npm start
```
